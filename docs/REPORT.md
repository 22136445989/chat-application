# Отчёт по практике

## 1. Выбранный проект

- **Название:** Real-time Chat
- **Ссылка из каталога:** *(вставить ссылку на каталог пет-проектов)*
- **Описание:** Веб-приложение для обмена сообщениями в реальном времени. Пользователи могут регистрироваться, входить в систему, создавать комнаты и обмениваться сообщениями.
- **Что реализовано:**
  - Регистрация и вход через Supabase Auth
  - Создание и просмотр комнат
  - Отправка и получение сообщений в real-time
  - Адаптивный интерфейс
  - Деплой на Vercel

## 3. Архитектура

### 3.1. Общая схема

```
[Пользователь]
     |
     v
[React + Vite]  <--->  [Supabase]
                            |
            +---------------+---------------+
            |               |               |
            v               v               v
         [Auth]        [Postgres]      [Realtime]
```

### 3.2. ERD (таблицы и связи)

```
auth.users ||--o{ profiles : id
profiles ||--o{ rooms : created_by
profiles ||--o{ messages : user_id
rooms ||--o{ messages : room_id
```

- **profiles** — профили пользователей
- **rooms** — комнаты чата
- **messages** — сообщения

### 3.3. Use Case

1. **Регистрация:** пользователь вводит имя, email и пароль → создаётся аккаунт и профиль.
2. **Вход:** пользователь вводит email и пароль → открывается список комнат.
3. **Создание комнаты:** пользователь вводит название → комната появляется в списке.
4. **Отправка сообщения:** пользователь выбирает комнату, пишет текст → сообщение отображается у всех участников в real-time.

### 3.4. API

Все запросы выполняются через клиент Supabase:

- `supabase.auth.signUp` — регистрация
- `supabase.auth.signInWithPassword` — вход
- `supabase.from('rooms').select()` — получение комнат
- `supabase.from('rooms').insert()` — создание комнаты
- `supabase.from('messages').select()` — получение сообщений
- `supabase.from('messages').insert()` — отправка сообщения
- `supabase.channel().on('postgres_changes')` — подписка на новые сообщения

## 7. Вывод по практике

В ходе практики было разработано приложение для обмена сообщениями в реальном времени. Были изучены и применены на практике React, TypeScript, Vite, Supabase и Vercel.
